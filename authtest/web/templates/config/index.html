Quick Start

Use the CLI to add IdPs from presets for common providers:

# Add Keycloak IdP

authtest config idp from-preset my-keycloak --preset keycloak --base-url https://keycloak.example.com --realm myrealm

# Add Okta IdP

authtest config idp from-preset my-okta --preset okta --okta-domain dev-123456.okta.com

# Add Azure AD IdP

authtest config idp from-preset my-azure --preset azure_ad --tenant-id contoso.onmicrosoft.com
